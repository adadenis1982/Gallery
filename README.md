# Gallery

Используется:

* JS
* React.js
* Redux
* Node.js
* ui библиотекa - fwt-internship-uikit

Использовать:
* npm i
* && npm start чтобы запустить приложение

Реализовано:
* верстка
* смена светлой и темной темы
* пагинация
* фильтр по строке
* фильтр по селекторам
